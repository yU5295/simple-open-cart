## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/dhatguy/PERN-Store.git
   ```
2. Install NPM packages in both server and client directory
   ```sh
   npm install
   ```
